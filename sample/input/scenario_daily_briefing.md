# Scenario: Daily Briefing Bot

## User profile
- Solo knowledge worker (PM / founder type)
- Uses Slack as primary IM
- Has Google Calendar, Gmail, Google Tasks
- US-based, English-speaking
- Non-technical (no coding skills, "vibe-coder")

## User says:

"I want a bot that sends me a daily briefing every morning in Slack. Like a personal assistant that tells me what my day looks like — meetings, to-dos, important emails, weather. I don't want to check 5 different apps every morning, I just want it pushed to me."

## What they DON'T know:
- What platform to build on
- Whether AI is needed or overkill
- How much it'll cost
- How long it'll take

## Expected MOAB behavior:
- Walk the honesty ladder (T0–T5)
- Surface existing tools first (T1)
- Show comparison table with ranges + source tiers
- Lock scope at ≤7 features
- Tag inferences
- Sign off with verification code
